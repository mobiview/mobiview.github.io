Frequently Asked Questions

Everything you need to know about using Mobile Viewer Io.

How do I test a website that requires a login? +

Since the viewer uses an iframe, you can log in directly within the frame if the target website allows iframe embedding and doesn't have strict "Same-Origin" policies.

Why does some content not load in the viewer? +

Some websites use X-Frame-Options: DENY or SAMEORIGIN headers for security, which prevents them from being displayed inside an iframe on another domain.

Can I test local development URLs like localhost? +

Yes! As long as your local server is running, you can enter http://localhost:port into the URL bar to test your designs before they go live.
